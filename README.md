# Wins Tracker 

This project is a Flutter-based mobile application designed to help users capture and reflect on their personal wins throughout the year. A “win” can be any positive moment, achievement, or progress, big or small.

# Key Features 

## Wins Logging
Add wins through text entries.
Voice to text or audio recording support for faster capture.

## Offline First 
Local storage for reliability and performance.

## Timeline View 
Chnonological display of recorded wins.
Easy Browsing and reviewing of past achivements.
